## 目標

依 CoreMain.md 的「任務核心」主題，對 WorkFlow 進行功能重設計：
1. 移除偏離主題的功能（Webhook、公開分享、多語言）

- **範圍**：功能重設計，含後端 API 調整 + 前端 UX 重構；不做新 DB migration（除 AI 升級）

- **暫緩**：F08 週報、F14 Emoji 反應、F16 個人效率分析

- **部署環境**：本機 Docker Compose（V3 已通過 E2E 20/20）
- **AI 升級前提**：Claude API Key 需使用者另行提供
